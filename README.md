# Trusting AI Agents: A Reinsurance Industry Case Study

This proof-of-concept application helps automate a data processing pipeline for the reinsurance industry. It parses excel submission packs, links to historical data, and produces a clean, final output. It features a multi-agent design with human-in-the-loop.
